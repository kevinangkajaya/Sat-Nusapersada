### How to Run Server
- Go to sat_nusapersada inner folder and type in command: `python manage.py runserver`.

### Login to Todo App
- Navigate to http://127.0.0.1:8000/todos/api
- Click login.
- Default credential for superuser:
    - User: asus
    - Password: password
